A set $\{v_i\}_{i\in \Lambda}$ in the [vector space](https://mathgloss.github.io/MathGloss/chicago/vector_space) $V$ is **orthogonal** if $\langle v_i, v_j\rangle =0$ for all $i\neq j$.
