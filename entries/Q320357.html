Closed set

MathGloss Definition

Let $X$ be a topological space. The subset $A \subset X$ is closed if its complement $X\setminus A$ is open. Equivalently, if $X$ is a metric space, then a subset $A \subset X$ is closed if and only if it contains all of its limit points.
