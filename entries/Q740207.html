isometry

MathGloss Definition

A function $f: X\to Y$ between metric spaces $(X, \rho)$ and $(Y,\sigma)$ is an isometry if for all $a,b \in X$, $$\sigma(f(a), f(b)) = \rho(a,b).$$ Two metric spaces are isometric if there exists an isometry between them.
