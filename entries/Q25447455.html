Lebesgue outer measure

MathGloss Definition

For any subset $A \subset\mathbb R^n$, the Lebesgue outer measure is given by $\mu^*(A) = \inf\{\sum\limits_{R\in\mathcal C}$vol$(R) \mid \mathcal C$ is a countable collection of rectangles covering $A\}$.
